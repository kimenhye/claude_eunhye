## 1. 개요

### 1.1 목적
터미널 디바이스의 메뉴, 사용자, 개인 데이터를 관리하는 기능을 설계한다.

### 1.2 범위
- 터미널 메뉴 관리
- 터미널 사용자 관리
- 터미널 개인 데이터 관리

---

## 2. 기능 설계

| No | 기능명 | 설명 | 우선순위 |
|----|--------|------|----------|
| 1 | 메뉴 관리 | 터미널 메뉴 구성/관리 | 상 |
| 2 | 사용자 관리 | 터미널 사용자 등록/관리 | 상 |
| 3 | 개인 데이터 | 터미널 개인 데이터 관리 | 중 |

---

## 3. API 설계

| No | Controller | 설명 |
|----|-----------|------|
| 1 | TerminalMenuController | 터미널 메뉴 관리 |
| 2 | TerminalUserController | 터미널 사용자 관리 |
| 3 | TerminalPersonalDataController | 터미널 개인 데이터 관리 |

## 4. 데이터 설계

- 별도의 데이터 계층 (legacy DAO 패턴 사용)
